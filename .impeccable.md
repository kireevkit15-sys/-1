## Design Context — РАЗУМ

### Users

Мужчины 18-45, интересующиеся саморазвитием, стратегическим мышлением, логикой. Технически грамотные (Telegram как основной вход).

**Контексты использования:**
- Утренняя разминка (5 минут, 5 вопросов — перед работой, поддержать стрик).
- Вечерние соревновательные батлы (15–30 минут, 1v1 «Осада крепости»).
- Обучение на ходу (в метро, на прогулке) — модули, AI-наставник.

**Job-to-be-done:** (1) прокачивать мозг через дуэли знаний, (2) расти по 5 статам (Стратегия, Логика, Эрудиция, Риторика, Интуиция) и видеть прогресс, (3) чувствовать себя элитой среди соперников.

### Brand Personality

**Три слова: отречённый, холодный, нерушимый.**

- **Отречённый** — убрано всё мягкое, милое, «дружелюбное». Продукт не заискивает. Он предъявляет стандарт — либо ты его держишь, либо нет. Ощущение монашеской кельи, не лобби кафе.
- **Холодный** — без тёплой сентиментальности, без «поздравляем!» и салютов. Эмоциональная температура низкая, даже если палитра тёплая по колориметрии. Точность библиотечного каталога, не объятия.
- **Нерушимый** — ощущение каменной крепости. Материалы — сталь, камень, кожа, бронза. Не пластик, не плюш, не бумага.

**Эмоции, которые вызывает интерфейс:** собранность, напряжение, уважение, гордость за заработанный прогресс. **Никогда:** эйфория, игривость, расслабленность, «дзынь!».

### Aesthetic Direction

**Тема:** dark-only для MVP. Светлая тема отложена post-launch — не подмешивать светлые surface'ы (`#fff`, `#e0e0e0`) даже в placeholder'ах.

**Две эстетики — не смешивать в одном экране:**

**1. Dark Academia — основной язык платформы (~85% экранов).**
Используется на: главная, логин, онбординг, профиль, лидерборд, разминка, обучение (`/learn/*`, `/learning/*`), AI-чат (`/chat`), feed, achievements, campaigns, referral, settings, about, публичный профиль, error/offline/404.

- **Палитра:** фон `#050505 → #222228`, медный `accent #CF9D7B` (brand), тёплый бежевый текст `#E8DDD3`, `accent-gold #B98D34`, `accent-bronze #B4783C`, `accent-silver #C0C0D2`, `accent-warm #724B39`. Ветки знаний (cyan/green/purple/orange/pink) — точечно, как маркеры категорий.
- **Эффекты:** sparse и медленные — `aurora-drift`, `breathe`, `reveal-up`, `reveal-from-dark`, мягкий `glow-pulse` на медном. `glass-card` и `liquid-glass` применяются целенаправленно (sticky bars, модалки, hero-карточки).
- **Шрифт:** Cormorant (`font-verse`) для цитат, заголовков-эпиграфов, вступлений к урокам. Inter (`font-sans`) для всего UI и body.
- **Форма:** скруглённые углы 12-16px (`rounded-2xl`, `rounded-xl`), тонкие 1px бордеры с `rgba(var(--color-accent)/0.15-0.25)`, глубокие мягкие тени.
- **Ощущение:** старая библиотека при свечах, монастырский скрипторий, кожаные переплёты. Тепло по колориметрии, но эмоционально отрешённо.

**2. Маскулинная агрессия — модальный режим (~15% экранов).**
Используется **только** на боевых и ритуальных экранах: `/battle/[id]` (все 5 фаз), `/battle/new` (matchmaking + VS), ритуальные стадии Barrier Flow в обучении (стадии с `cold-steel`/`cold-blood` палитрой).

- **Палитра:** добавляются `cold-blood #8B2E2E`, `cold-steel #6B7D8C`, `cold-steel-dim #3A4550`, CTA `.cta-battle` (red→orange→gold gradient), `accent-red #C0392B`, `accent-orange #E67E22`. Базовый фон тот же dark (#050505), но бордеры и тени смещаются к красно-стальному.
- **Эффекты:** резкие и быстрые — `shake`, `particle-burst`, `blood-pulse`, `pulse-critical`, `steel-shimmer`, `text-flicker`, `battle-slam`, `battle-scale-in`. Допускается `glow-pulse` на золотом/красном.
- **Шрифт:** Inter с тяжёлыми весами (700-900), uppercase-метки с `tracking-[0.05-0.1em]`. Cormorant убран.
- **Форма:** прямые углы (`rounded-none` или `rounded-sm`), остроугольные badges, жёсткие линии, толстые бордеры 2px в ключевых местах (HP-бары, difficulty tiers).
- **Ощущение:** арена гладиаторов, дуэль, осада крепости. Холод стали, не уют.

**Правило переключения:** эстетика определяется маршрутом, а не компонентом. UI-примитивы (`Button`, `Card`, `Toast`, `EmptyState`, `Skeleton`) по умолчанию — Dark Academia. На боевых экранах они получают MA-вариант через prop или контекст (например, `<Button variant="battle">`). Примитивы не меняют свою DA-форму «на всякий случай» — они сохраняют мягкий радиус и медный акцент везде, кроме явного MA-режима.

- Старые библиотеки, монастырские скриптории, кабинеты-study
- Финтех dark premium: Revolut, Robinhood dark
- Фитнес-трекеры premium: Whoop, Oura
- Chess.com (шахматная атмосфера)
- Игровые лобби — **только для боевых экранов**: Valorant, League of Legends

**Антиреференсы:**
- Пастель, cyan-on-dark gradient, neon-purple-to-blue (generic AI palette)
- Скругления «ради скругления», декоративные стикеры, эмодзи в UI
- Gradient text (`background-clip: text` на градиенте) — кроме узкого `.text-metallic` для логотипа, под ревью
- Casual mobile game feel: cute fonts, bouncy springs, confetti particles
- Glassmorphism везде — glass применяется точечно (sticky bars, модалки, hero), не декоративно
- Border-left/right `> 1px` как цветной акцент-стрип (absolute ban)
- Duolingo-style визуал (механики ок, визуал — нет)

### Design Principles

1. **Функция прежде украшения.** Каждый эффект, цвет, анимация обязан решать задачу (focus, feedback, hierarchy, state change). Декоративность — повод удалить. «Красиво ради красиво» не принимается.

2. **Контраст через аскетизм.** Большие поля пустоты, редкие акценты. Цвет = событие, не состояние по умолчанию. Если «красиво везде» — значит, нигде.

3. **Цвет веток — единственный цветной язык за пределами медного.** За пределами веток (strategy/logic/erudition/rhetoric/intuition), медного акцента и (на боевых экранах) cold-blood/cold-steel — только нейтрали. Никаких случайных hex-значений. Все hex'и — через CSS-переменные в `globals.css`.

4. **Тяжесть материалов.** Интерфейс должен ощущаться из камня/стали/кожи, не из пластика. Тени глубокие, бордеры тонкие, типографика с весом (Inter 600–900 для заголовков на UI; Cormorant — только для верстальных акцентов). Избегать воздушных, «парящих» компонентов без веса.

5. **Маскулинная агрессия — оружие, а не костюм.** Не «везде должно быть грозно» — иначе приём теряет силу. Боевая эстетика включается только там, где идёт бой. В повседневных экранах (дом, профиль, обучение) — Dark Academia держит тон: собранный, отрешённый, библиотечный. Контраст DA → MA при входе в батл даёт тот самый удар, ради которого всё и делается.
