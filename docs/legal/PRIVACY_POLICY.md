# Политика конфиденциальности — РАЗУМ

**Дата вступления в силу:** 2026-04-21
**Последнее обновление:** 2026-04-21

Данный документ — черновик, подготовленный командой разработки. Перед публичным запуском обязательна юридическая проверка у профильного юриста (GDPR/ФЗ-152 на РФ).

## 1. Кто мы

РАЗУМ — PWA-платформа для саморазвития мужчин через RPG-батлы знаний, систему обучения и AI-наставника. Оператор обработки персональных данных — **владелец проекта РАЗУМ** (далее «мы», «оператор»). Контакт: `privacy@razum.app`.

## 2. Какие данные мы собираем

### 2.1 Данные, предоставленные пользователем

- **Регистрация через Telegram:** Telegram user ID, first_name, last_name (если задано), username, photo_url (аватар), language_code.
- **Регистрация через email:** email, пароль (хранится только bcrypt-хеш), отображаемое имя.
- **Профиль:** возраст-диапазон, самоописание (добровольное, сохраняется при прохождении определения уровня).
- **Ответы в обучении и батлах:** выбранные варианты, развёрнутые ответы на вопросы барьеров, диалог с AI-наставником.
- **Метрики прогресса:** очки характеристик (Ум/Душа/Воля/Навык/Вкус), ранг, серии (streak), пройденные концепты.

### 2.2 Данные, собираемые автоматически

- **Техническая телеметрия:** IP-адрес, User-Agent, версия браузера, тип устройства, экран (для адаптации UI).
- **Логи взаимодействия:** время входа, время активности, страницы, клики на кнопки CTA (для улучшения UX, не для таргетинга).
- **Ошибки приложения:** стек-трейсы через Sentry (см. §4.2).
- **Cookie:** `razum_session` (HTTP-only, Secure, SameSite=Lax) — идентификатор сессии; `razum_csrf` — CSRF-защита.

### 2.3 Что мы НЕ собираем

- Финансовые данные (платёжных функций в MVP нет).
- Точную геолокацию (только страна из IP для санкционного комплаенса).
- Данные из сторонних соцсетей кроме Telegram.
- Биометрию.

## 3. Для чего используем

## 4. Кому передаём

Мы **не продаём** данные третьим лицам. Передача происходит только:

### 4.1 Процессорам (обработчикам по поручению)

- **Anthropic (PBC, США)** — отправляем содержание твоих ответов на вопросы обучения и диалогов с AI-наставником для генерации ответов через Claude API. Не используется для тренировки моделей (Anthropic DPA). Данные хранятся у Anthropic до 30 дней для абьюз-монитоинга.
- **Telegram (Великобритания/BVI)** — только если ты зарегистрирован через Telegram. Мы получаем только те поля, что ты разрешил при логине.
- **Функциональная инфраструктура (VPS-провайдер в ЕС):** хостинг БД и приложения. Обработчик отвечает требованиям GDPR.
- **Sentry (Functional Software, Inc.)** — стек-трейсы ошибок. Мы включили фильтры, которые вычищают Authorization-хедер, cookie и email из событий (`beforeSend` хук). Оставшиеся данные — URL, тип устройства, стек JS.
- **AWS S3 (`eu-central-1`)** — зашифрованные резервные копии БД.

Список актуальных процессоров публикуется на `/privacy/processors` (обновляется при добавлении новых).

### 4.2 Государственным органам

Только по официальному запросу и в объёме, предусмотренном законом юрисдикции оператора. Мы уведомляем пользователя, если это не запрещено запросом.

## 5. Как защищаем

- TLS 1.2+ везде, HSTS preload.
- Пароли — bcrypt (10 раундов).
- Секреты (JWT, API-ключи) не логируются, ротация раз в 90–180 дней (см. `docs/SECRETS_ROTATION.md`).
- CSP, X-Frame-Options: DENY, Permissions-Policy.
- Шифрование S3-бэкапов (SSE-S3).
- Принцип минимизации: в Sentry не уходят email/cookie (см. `sentry.client.config.ts:17`).

## 6. Твои права (GDPR ст. 15–22)

- **Доступ:** `/settings/data/export` — JSON со всем, что о тебе сохранено.
- **Исправление:** редактируй профиль в `/settings/profile`.
- **Удаление (right to be forgotten):** `/settings/data/delete` — аккаунт анонимизируется в течение 30 дней, тренировочные данные отвязываются от идентификаторов. Бэкапы перезаписываются естественным ротированием за 30 дней.
- **Портабельность:** тот же `/export` даёт машиночитаемый JSON.
- **Возражение:** напиши на `privacy@razum.app` — прекратим обработку, для которой основание «законный интерес».
- **Отзыв согласия:** для Telegram-уведомлений — команда `/stop` в боте.
- **Жалоба:** в надзорный орган по месту жительства (для РФ — Роскомнадзор).

Срок ответа на запрос — 30 дней.

## 7. Несовершеннолетние

Сервис предназначен для лиц старше **16 лет**. Мы не проверяем возраст технически, но оставляем за собой право удалить аккаунт при обнаружении несоответствия. Родители/опекуны могут запросить удаление данных через `privacy@razum.app`.

## 8. Международная передача

Данные обрабатываются в ЕС (VPS) и США (Anthropic, Sentry). Для передач США используются стандартные договорные условия ЕС (SCCs).

## 9. Изменения

О существенных изменениях политики уведомляем через `/notifications` и баннер в приложении за 14 дней до вступления в силу. Продолжение использования после даты вступления означает согласие.
