# РАЗУМ — Дорожная карта MVP

## Обзор

**Цель MVP:** Валидировать core-гипотезу — «мужчины находят соревновательные интеллектуальные батлы достаточно увлекательными, чтобы возвращаться ежедневно».

**Срок:** 3 месяца (12 недель)
**Команда:** 3 разработчика
**Результат:** 500 бета-тестеров используют продукт ежедневно

---

## MVP = 8 фичей

| # | Фича | Приоритет |
|---|------|-----------|
| 1 | Вход через Telegram (+ email) | P0 |
| 2 | Ежедневная разминка (5 вопросов) | P0 |
| 3 | 1v1 батл против бота и человека | P0 |
| 4 | 2 ветви обучения (Стратегия + Логика) | P1 |
| 5 | 1 AI-диалог в день (сократический метод) | P1 |
| 6 | Профиль с радар-чартом 5 статов | P1 |
| 7 | Стрик-счётчик | P2 |
| 8 | Лидерборд | P2 |

---

## Неделя 1-2: Фундамент

### День 1-2: Инициализация монорепо
- [ ] Turborepo + apps/web + apps/api + packages/shared
- [ ] ESLint, Prettier, TypeScript strict mode
- [ ] Docker Compose: PostgreSQL 16 + Redis 7
- [ ] GitHub Actions: lint + typecheck + test
- [ ] Prisma schema + первая миграция

### День 3-4: Аутентификация
- [ ] NextAuth.js v5 + Telegram Login Widget
- [ ] Email/пароль как запасной вход
- [ ] JWT tokens + refresh
- [ ] Таблица User + UserStats (авто-создание при регистрации)

### День 5-7: Ядро батл-движка
- [ ] `packages/shared` — стейт-машина батла (чистый TS)
- [ ] Состояния: WAITING → CATEGORY_SELECT → ATTACK → DEFENSE → SWAP → RESULT
- [ ] Скоринг по сложности (bronze/silver/gold)
- [ ] Unit-тесты на все переходы состояний

### День 8-10: Батл API + бот
- [ ] NestJS BattleModule + BattleGateway (Socket.IO)
- [ ] Бот-противник (детерминированный, 60% правильных)
- [ ] REST: POST /battles, GET /battles/:id
- [ ] WS: battle:start → battle:round → battle:answer → battle:result

### День 11-14: Батл UI
- [ ] Next.js: /battle/new, /battle/[id]
- [ ] Экраны: выбор категории → атака → защита → результат
- [ ] Таймер 60 сек на раунд
- [ ] Mobile-first, тёмная тема
- [ ] PWA manifest + service worker

**Milestone:** Вход через Telegram → батл с ботом → результат на мобильном.

---

## Неделя 3-4: Мультиплеер + статы

### День 15-18: Real-time 1v1
- [ ] Redis SORTED SET для матчмейкинга по рейтингу
- [ ] Socket.IO rooms для каждого батла
- [ ] Дисконнект >30 сек = авто-поражение
- [ ] Таймаут раунда 60 сек = авто-ответ

### День 19-21: Аватар и статы
- [ ] 5 статов привязаны к вопросам
- [ ] level = floor(sqrt(xp / 100))
- [ ] Страница профиля с радар-чартом (Recharts)
- [ ] История батлов на профиле

### День 22-25: Контент
- [ ] Админ API: POST /questions/bulk (JSON)
- [ ] Скрипт генерации через Claude API → JSON
- [ ] Seed: 200 вопросов (Стратегия + Логика)
- [ ] Ручная ревизия каждого вопроса

### День 26-28: Первый деплой
- [ ] VPS: Docker Compose + nginx + Let's Encrypt
- [ ] Sentry (free tier)
- [ ] Тест с 5-10 реальными людьми

**Milestone:** Два человека играют друг против друга. Статы растут. Профиль работает.

---

## Месяц 2: Core Loop

### Неделя 5-6: Модули обучения
- [ ] Таблицы Module + UserModuleProgress
- [ ] GET /modules?branch= + GET /modules/:id
- [ ] Плоский список модулей по 2 веткам
- [ ] Модуль = заголовок + описание + 5 вопросов
- [ ] Прогресс-бар, последовательная разблокировка

### Неделя 6-7: AI сократический диалог
- [ ] Anthropic SDK интеграция
- [ ] Системный промпт для сократического метода
- [ ] POST /ai/dialogue, GET /ai/dialogue/:id
- [ ] Rate limit: 1/день free (Redis TTL)
- [ ] UI: чат на странице модуля
- [ ] Макс 2000 output tokens, 10 обменов

### Неделя 7-8: Daily engagement
- [ ] GET /warmup/today — 5 случайных вопросов
- [ ] POST /warmup/submit — подсчёт + стрик
- [ ] Push-уведомления (Web Push API)
- [ ] Главный экран: разминка + батл + стрик + статы

**Milestone:** Полный core loop: учись → тренируйся → соревнуйся → AI-коучинг.

---

## Месяц 3: Бета-продукт

### Неделя 9-10: Контент x2.5
- [ ] 500+ вопросов по 2 веткам (6 категорий)
- [ ] AI-генерация + админ-панель для ревизии
- [ ] Жалобы на плохие вопросы (user reports)

### Неделя 11-12: Социал
- [ ] Лидерборд: топ-20 за неделю
- [ ] «Вызови друга» — инвайт-ссылка через Telegram
- [ ] Шеринг профиля (OG-картинка)
- [ ] Класс мыслителя (по доминирующему стату)

### Неделя 12: Запуск беты
- [ ] Lighthouse >90
- [ ] k6 нагрузочное тестирование (100 батлов)
- [ ] Политика конфиденциальности + соглашение
- [ ] Telegram-канал community
- [ ] Инвайт 200-500 бета-тестеров

**Milestone:** 500 человек используют продукт ежедневно.
